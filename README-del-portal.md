# Guia de Uso del Portal Municipal

Este documento ha sido preparado como una guia simple de demostracion para la municipalidad. Explica como ingresar al portal, que puede revisar cada perfil y cuales son las credenciales de prueba disponibles.

## 1. Objetivo del portal

El portal permite separar claramente dos accesos:

- `Contribuyente`: para consultar obligaciones, revisar historial y realizar pagos
- `Administracion`: para revisar metricas, transacciones recientes y seguimiento general del sistema

## 2. Tipos de acceso

En la pantalla principal se muestran dos opciones:

- `Contribuyente`
- `Administración`

Cada una dirige a un acceso diferente y con credenciales independientes.

## 4. Ingreso como contribuyente

Para entrar al panel ciudadano:

1. Selecciona la opcion `Contribuyente`.
2. Escribe el numero de DNI.
3. Escribe la contraseña de cartilla municipal.
4. Presiona `Ingresar al panel ciudadano`.

## 5. Ayuda para ubicar la contraseña

Debajo del campo de contraseña aparece un mensaje de ayuda.

Si el usuario no sabe donde encontrar la clave:

1. Haz clic en el icono de interrogacion.
2. Se mostrara una imagen de referencia de la cartilla municipal.
3. Usa esa imagen para identificar la ubicacion de la contraseña.

## 6. Funciones del panel contribuyente

Dentro del acceso ciudadano se puede:

- Ver el resumen de deuda
- Revisar obligaciones pendientes o vencidas
- Consultar pagos recientes
- Seleccionar conceptos para pago
- Completar el flujo de pago con los metodos habilitados

## 7. Flujo basico de pago

Para realizar una prueba de pago:

1. Inicia sesion como contribuyente.
2. Entra a `Mis Obligaciones`.
3. Selecciona una o varias obligaciones.
4. Continúa al flujo de pago.
5. Elige el metodo disponible.
6. Completa la simulacion del proceso.

Luego del pago:

- las obligaciones pagadas se actualizan
- el historial del contribuyente cambia
- el resumen de deuda se refresca

## 8. Ingreso como administración

Para entrar al panel administrativo:

1. Selecciona la opcion `Administración`.
2. Escribe el usuario `admin`.
3. Escribe la contraseña `admin123`.
4. Presiona `Ingresar al panel administrativo`.

## 9. Funciones del panel administrativo

Dentro del acceso administrativo se puede visualizar:

- Indicadores generales de recaudacion
- Transacciones recientes
- Distribucion de pagos por tipo
- Seguimiento general de la actividad del portal

## 10. Relacion entre ambos paneles

La demostracion esta preparada para que los movimientos del contribuyente se reflejen en la vista administrativa.

Por ejemplo:

- si un contribuyente realiza un pago, el historial ciudadano se actualiza
- ese mismo movimiento puede verse luego en el panel administrativo

## 11. Interaccion con la pasarela Pay-Me

Durante el proceso de pago, el portal se conecta con la pasarela Pay-Me para mostrar la experiencia de cobro dentro del mismo flujo del contribuyente.

En esta demostracion, Pay-Me aporta lo siguiente:

- Presenta una experiencia de pago integrada dentro del portal
- Permite mostrar distintos metodos de pago en una sola vista
- Da continuidad al proceso sin que el contribuyente sienta que sale del portal
- Ayuda a representar una experiencia de pago mas ordenada, moderna y guiada

Para la municipalidad, esto aporta valor porque:

- mejora la experiencia de atencion digital al contribuyente
- facilita la comprension del proceso de pago
- permite mostrar una solucion de recaudo mas completa
- refuerza la idea de trazabilidad entre el pago realizado y su reflejo en el panel administrativo

## 12. Recomendaciones para la demostracion

- Primero ingresa como contribuyente y realiza una prueba de pago
- Luego ingresa como administrador para validar el cambio en el panel interno
- Usa el icono de ayuda si deseas explicar visualmente la cartilla municipal
- Cierra sesion al terminar cada prueba

## 13. Cierre de sesión

Tanto el contribuyente como el administrador pueden salir del sistema desde la opcion de cierre de sesion ubicada en la parte superior del portal.

## 14. Uso sugerido en presentaciones

Si esta guia se usa para mostrar el portal a un equipo municipal, se recomienda el siguiente orden:

1. Mostrar la pantalla inicial con la separacion de accesos.
2. Explicar el ingreso del contribuyente con DNI y contraseña de cartilla.
3. Enseñar la ayuda visual del icono de interrogacion.
4. Revisar obligaciones y simular un pago.
5. Ingresar al panel administrativo.
6. Mostrar como se reflejan los cambios del flujo ciudadano.
